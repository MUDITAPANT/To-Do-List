# To-Do-List
A simple and responsive To-Do List web app built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Add task details
- Set task priority: Low, Medium, High
- Add due date and time
- Mark tasks as completed
- Edit tasks
- Delete tasks
- Search tasks using search button
- Progress bar to track completed tasks
- Empty task image when no tasks are added
- Data saved in browser local storage
- Confetti animation when all tasks are completed

## Technologies Used

- HTML
- CSS
- JavaScript
- Font Awesome Icons
- Local Storage

How to Use
Enter a task name.
Select priority.
Choose a due date and time.
Add task details.
Click the plus button to add the task.
Click a task name to view its details.
Use the edit button to update a task.
Use the delete button to remove a task.
Use the search box and search icon to find a task.
